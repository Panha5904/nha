Recommended Course Paths

Foundational Courses

Where to start

Core languages and concepts to build a strong base in programming.

  • Intro to Programming — Python or JavaScript
  • Web Fundamentals — HTML & CSS
  • OOP — Java/C#
  • DSA — Data Structures & Algorithms

Web Development

Frontend & Backend

Paths to become a web developer — split between client and server.

  • Front-end: JavaScript, React, TypeScript
  • Back-end: Node.js, Python (Django/Flask), Go
  • Recommended outcome: Full-stack projects

Mobile Development

Native & cross-platform

Build mobile apps for iOS, Android, or both.

  • iOS: Swift / SwiftUI
  • Android: Kotlin or Java
  • Cross-platform: React Native or Flutter (Dart)

Data Science & AI

Analysis & modeling

From data cleaning to building machine learning models.

  • Data Analysis: Python with Pandas, NumPy, SQL
  • Machine Learning: scikit-learn, TensorFlow, PyTorch
  • Outcome: Data projects and model demos

Systems & Performance

Low-level & efficient

For when speed, memory control, and hardware matter.

  • C & C++: OS, game engines, embedded
  • Go / Rust: Concurrent systems, cloud services
  • Recommended: CS50 and system programming courses

Quick Tips for Success

  • Start small: build tiny projects and iterate.
  • Practice consistently: short daily practice beats long infrequent sessions.
  • Build a portfolio: showcase projects on GitHub and a simple site.
